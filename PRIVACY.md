## 1. Overview

OpenQuests is a lightweight, community-driven strategy RPG operated as an experimental online game.

We value simplicity and transparency. This Privacy Policy explains what data we collect, how we use it, and what we do not do with it.

---

## 2. Information We Collect

OpenQuests may collect the following information:

### Account & Identification Data
- GitHub username
- GitHub user ID
- Issue number (used as player identifier)

### Gameplay Data
- Character name, class, level, XP
- Clan affiliation
- In-game actions (gather, explore, attack)
- Resources (food, wood, gold)
- Achievements and titles
- Game statistics (wins, losses, encounters, etc.)

### Technical Data
- Basic request logs (via hosting provider such as Cloudflare)
- IP address (processed by hosting infrastructure, not directly stored by us unless required for security)

---

## 3. How We Use Information

We use collected data strictly to:

- Operate and maintain the game
- Process daily world updates
- Generate world and location logs
- Enable clan and world chat (if applicable)
- Improve gameplay balance and mechanics
- Prevent abuse or cheating

We do **not**:
- Sell personal data
- Share personal data with advertisers
- Use data for marketing purposes

---

## 4. AI-Generated Content

OpenQuests may use AI services to generate:
- World narration
- Location summaries
- Event storytelling

Gameplay data (such as event summaries or statistics) may be sent to third-party AI providers solely for narrative generation. No sensitive personal data is intentionally transmitted.

---

## 5. Data Storage

Game data is stored using:
- GitHub repositories
- Cloudflare infrastructure
- Serverless storage systems

Data may be publicly visible if stored in public repositories.

Players should not submit personal or sensitive information in:
- Character backstories
- Chat messages
- Public issues

---

## 6. Data Retention

Gameplay data may be stored indefinitely as part of the persistent world history.

If you would like your player data removed, you may:
- Contact the repository owner

Complete erasure may not be possible if data exists in public Git history.

---

## 7. Security

We take reasonable measures to protect game integrity, but OpenQuests is an experimental open project.

No system is guaranteed to be 100% secure.

---

## 8. Children's Privacy

OpenQuests is not intended for children under 13.

If you are under 13, please do not use this service.

---

## 9. Changes to This Policy

This policy may be updated as the project evolves. Continued use of OpenQuests constitutes acceptance of changes.

---
